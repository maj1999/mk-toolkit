# mk-toolkit 变更日志

整个项目（`mk-toolkit` 插件）的版本演进。版本号遵循 [SemVer](https://semver.org/lang/zh-CN/)，与 git tag `vX.Y.Z` 及 `.claude-plugin/marketplace.json` 的 `plugins[0].version` 保持一致。

每次发布聚合「上次 tag 以来各 skill 的变更」。各 skill 自己的细粒度历史见其目录下的 `CHANGELOG.md`。发布与版本规范见 [MAINTAINING.md](./MAINTAINING.md)。
